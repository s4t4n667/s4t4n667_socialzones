# s4t4n667_socialzones
## Zone-based social payment system 💸👥

Specify zones in which players can earn money whilst socialising!

- Supports ESX, QBCore and Qbox.
- Supports ox_inventory.
- Locales: English.

**Documentation:** [Click Here](https://s4t4n667.gitbook.io/asgaard-developments/free-scripts/s4t4n667_socialzones)
<br>

> Dependencies:
> <br>
> [ox_lib](https://github.com/overextended/ox_lib)
> <br>
> [ox_inventory](https://github.com/overextended/ox_inventory)
> <br>
> [PolyZone](https://github.com/mkafrin/PolyZone)

I recommend using [sd-zonecreator](https://github.com/Samuels-Development/sd-zonecreator) from Samuels-Developments to help you create your zones!
